CartoDB Night Lights Basemap

A dark global basemap with "lights" in the oceans

This basemap exemplifies what John Pickles (2004) defined as a cartographic gaze, a miniature version of the world designed for a specific purpose. This particular map is best used as a backdrop for displaying major land features at a global scale. Since the map uses a Mollweide projection, data areas will be accurate across the map, but data shapes will appear distorted in polar regions. For best results, use this basemap to show generalized, global data where feature area is more important than feature shape, and avoid displaying data that would cover up the characteristic white "lights" in the ocean ridges.
